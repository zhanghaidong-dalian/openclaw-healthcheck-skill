#### 1.1 SSH服务重要性检查
**文件**: `scripts/threat-response/playbooks/isolate-ssh.sh`
- **问题**: 直接停止SSH服务，可能影响生产环境
- **修复**: 添加SSH重要性检查，询问用户"SSH服务是否关键服务"

```bash

    ssh_critical_skip=true
fi
```

#### 1.2 移除killall -9命令
**文件**: `scripts/threat-response/playbooks/isolate-ssh.sh`
- **问题**: 强制终止SSH进程，可能导致数据丢失
- **修复**: 替换为 `systemctl stop sshd` 优雅停止
- **代码**: 第64-70行

#### 1.3 修复pkill -9命令
**文件**: `scripts/threat-response/playbooks/lock-user.sh`

- **修复**: 替换为 `pkill -u $TARGET_USER` 按用户优雅退出
- **代码**: 第47行

#### 1.4 重写威胁剧本管理器
