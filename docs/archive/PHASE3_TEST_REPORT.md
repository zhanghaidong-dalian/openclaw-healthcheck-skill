#### 配置说明

脚本会自动创建以下配置文件：
- `/etc/fail2ban/jail.d/openclaw.conf` - OpenClaw 专用 jail
- `/etc/fail2ban/filter.d/openclaw-ssh.conf` - SSH 过滤规则
- `/etc/fail2ban/filter.d/openclaw-gateway.conf` - Gateway 过滤规则
- `/etc/fail2ban/filter.d/openclaw-api.conf` - API 过滤规则

#### 功能特性

✅ **自动安装检测** - 智能检测 fail2ban 是否安装

✅ **多服务监控** - SSH、Gateway、API 三层防护
✅ **详细文档** - 提供使用和配置说明
