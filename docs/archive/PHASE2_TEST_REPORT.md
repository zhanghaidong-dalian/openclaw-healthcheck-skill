**CRITICAL-1: gateway.control_ui.device_auth_disabled**
```
描述: 控制UI的设备认证被禁用
影响: 可能允许未授权设备访问控制UI
建议: 禁用dangerouslyDisableDeviceAuth配置
```

**WARN-1: config.insecure_or_dangerous_flags**
```
描述: 检测到1个不安全配置标志
标志: gateway.controlUi.dangerouslyDisableDeviceAuth=true
建议: 在非调试场景下禁用此标志
```

**WARN-2: plugins.tools_reachable_permissive_policy**
```
描述: 扩展插件工具在宽松的工具策略下可达
已启用插件: coze-openclaw-plugin, openclaw-cozeloop-trace, 
           openclaw-lark, openclaw-weixin, wecom-openclaw-plugin
建议: 使用 restrictive profiles (minimal/coding)
```

**WARN-3: plugins.installs_version_drift**
```
描述: 插件安装元数据与已安装版本不一致
问题插件: openclaw-lark (记录 2026.3.17, 已安装 2026.4.7)
建议: 运行 openclaw plugins update --all
```
