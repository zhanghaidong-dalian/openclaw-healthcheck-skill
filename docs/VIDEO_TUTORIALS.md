## 📖 快速入门视频笔记

### 📹 视频1: 5分钟快速入门

**学习目标**: 掌握基础使用流程

**关键步骤**:

```
1️⃣ 触发检查
   ↓
   "帮我检查 OpenClaw 安全"
   ↓
2️⃣ 查看结果
   ↓
   安全评分: 95/100
   问题数: 3个
   ↓
3️⃣ 执行修复
   ↓
   选择要修复的项目
   ↓
4️⃣ 完成
```

**命令速记**:
```bash
# 一键检查
bash <path>/scripts/quick-check.sh

# 深度检查
openclaw security audit --deep

# 自动修复
bash <path>/scripts/auto-fixer-v5.sh
```

### 📹 视频5: 定时任务

**学习目标**: 设置自动化安全巡检

**Cron 表达式**:

```bash
# 每天早上8点
0 8 * * *

# 每周一早上9点
0 9 * * 1

# 每天早中晚三次
0 8,12,18 * * *
```

**配置示例**:

```
AI: "每天早上8点检查一次安全，周一深度检查"
↓
AI 自动配置:
- 日常检查: 0 8 * * *
- 深度检查: 0 9 * * 1
```

**启动监控**:

```bash

# 持续监控
python3 agent/realtime-monitor.py --interval 60

# 导出报告
python3 agent/realtime-monitor.py --export alert.json
```
