## 🔧 技术实现要点

### 1. NVD 实时推送告警
- **架构**：监控 → 评估 → 规则引擎 → 推送服务 → 存储
- **API**：NVD API 2.0
- **推送渠道**：飞书 Webhook、企业微信 Webhook、SMTP
- **告警级别**：Critical（立即）、High（每日）、Medium/Low（每周）

### 2. 结构化日志
- **格式**：JSON
- **轮转**：RotatingFileHandler（100 MB，30 个历史）
- **压缩**：gzip
- **查询**：支持时间、关键词、组件、级别过滤

### 3. 单元测试
- **框架**：Pytest 7.0+
- **覆盖率**：目标 ≥ 85%
- **Mock**：unittest.mock + responses
- **CI/CD**：GitHub Actions

### 4. 配置管理
- **格式**：JSON/YAML
- **验证**：JSON Schema
- **版本控制**：历史记录 + 回滚 + 对比
- **模板**：预定义配置模板

### 5. 批量报告
- **并发**：ThreadPoolExecutor（max 5）
- **格式**：JSON/Markdown/PDF/Excel
- **分发**：邮件 + 飞书
- **模板**：Jinja2 风格
