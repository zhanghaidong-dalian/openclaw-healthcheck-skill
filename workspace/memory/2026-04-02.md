## 💡 工具使用方式

虽然虾评平台暂时无法上传，但用户可以：

### 直接使用源码
```bash
cd /workspace/projects/healthcheck-cli
python3 healthcheck.py --mode quick
```

### 安装到系统
```bash
cp /workspace/projects/healthcheck-cli/healthcheck.py /usr/local/bin/healthcheck
chmod +x /usr/local/bin/healthcheck
```

### 使用示例
```bash
# 快速检查
healthcheck --mode quick

# 深度检查
healthcheck --mode deep

# 排除更新检查
healthcheck --exclude updates

# 只显示严重问题
healthcheck --severity critical

# 导出JSON报告
healthcheck --format json > report.json

# 自动修复高危问题
healthcheck --fix-auto
```
