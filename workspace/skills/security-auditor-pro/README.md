## 技能简介

Security Auditor Pro 是一款专为OpenClaw用户设计的安全审计技能，帮助您在安装第三方技能前进行全面的安全检测，防止恶意代码和潜在风险。

## 核心功能

### 🆓 免费版

- **安装前扫描**: 检测恶意代码、危险权限
- **风险评分**: 0-100分安全评分
- **30+检测规则**: 覆盖高/中/低风险
- **详细报告**: 清晰的审计结果和建议

### 💼 专业版 (¥1,999/年)

- **运行时监控**: 实时监控敏感操作
- **飞书/企微告警**: 实时推送安全通知
- **自动报告**: 每日/每周/每月自动生成
- **API对接**: 支持企业系统集成

### 🏢 企业版 (¥19,999/年)

- **多用户管理**: 团队协作支持
- **私有威胁库**: 企业自定义规则
- **合规审计**: 符合等保/ISO标准
- **专属支持**: 7×24小时技术支持

## 检测能力

### 🔴 高风险 (-25分/项)

| 规则 | 说明 |
|------|------|
| HR001 | SSH密钥目录访问 |
| HR002 | AWS凭证访问 |
| HR003 | GPG密钥访问 |
| HR004 | Shell命令执行 |
| HR005 | 动态代码执行 |
| HR006 | Base64数据外发 |
| HR007 | 可疑域名数据传输 |

### 🟠 中风险 (-10分/项)

| 规则 | 说明 |
|------|------|
| MR001 | 环境变量读取 |
| MR002 | 配置文件读取 |
| MR003 | 代码混淆检测 |
| MR004 | 文件写入操作 |
| MR005 | 子进程创建 |

### 🟡 低风险 (-5分/项)

| 规则 | 说明 |
|------|------|
| LR001 | 用户目录访问 |
| LR002 | 文件系统操作 |
| LR003 | 进程信息读取 |
| LR004 | 日志记录 |

## 风险评分标准

```
90-100分 🟢 安全 - 可以放心安装
70-89分  🟡 注意 - 建议关注
50-69分  🟠 谨慎 - 谨慎安装
0-49分   🔴 危险 - 不建议安装
```
