# 🍀 大乐透杀号选球技能

## 概述

基于本地数据库中的历史数据，执行多种杀号规则过滤号码，剩余号码生成预测组合。

**数据源**：优先从本地数据库读取（复用 super-lotto 技能的数据），数据不足时提示用户抓取。

## 杀号规则

| # | 规则名称 | 逻辑说明 |
|---|----------|----------|
| 1 | 冷热号杀号 | 频率超过平均值2倍的热号 + 频率最低的冷号 |
| 2 | 遗漏值杀号 | 前区遗漏≥15期、后区遗漏≥10期 |
| 3 | 尾数杀号 | 热尾（频率>1.5倍均值的尾数全部杀）+ 冷尾（连续5期未出） |
| 4 | 断区杀号 | 7区划分后，出号密度极低的区间整体杀掉 |
| 5 | 加减法杀号 | 上期相邻差 +3/-3运算 |
| 6 | 公式杀号 | 三种经典公式：A5-A3-A1, A4-A2等 |
| 7 | 首尾联动 | (最小前区+最大后区)%35 |
| 8 | 和值杀号 | 根据近10期和值趋势判断 |

**高置信度**：被≥2规则同时命中的号码，杀号可信度更高。

## 快速开始

### 1. 杀号预测（默认3组）

```bash
python main.py
```

### 2. 指定分析期数和预测组数

```bash
# 用50期数据分析，生成5组预测
python main.py --history 50 --count 5
```

### 3. 仅查看杀号结果

```bash
python main.py --kill-only
```

### 4. 运行回测验证

```bash
python main.py --backtest
```

### 5. 查看规则说明

```bash
python main.py --rules
```

## 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--history`, `-n` | 分析的历史期数 | 30 |
| `--count`, `-c` | 生成预测组数 | 3 |
| `--kill-only`, `-k` | 只显示杀号结果 | False |
| `--backtest`, `-b` | 运行回测验证 | False |
| `--rules`, `-r` | 显示规则说明 | False |
| `--verbose`, `-v` | 显示详细信息 | False |
